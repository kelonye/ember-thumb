Install
---

    $ component install kelonye/ember-thumb

Usage
---

```javascript

App.ThumbView = Em.View.extend(require('ember-thumb'), {
  width: 100, //px
  height: 100, //px
  src: 'photo.jpg',
});

```


Example
---

    $ make example
